# Chroma

A playful **color studio** to pick colors and convert between **HEX** and **RGB**.
Built with **Next.js**, **TypeScript**, and **Tailwind CSS**, and deployed automatically to **GitHub Pages**.

## Features

- 🎨 **Interactive color wheel** — pick a color and see HEX + RGB update live
- 🎲 **Surprise me!** — generate a random color (and a fresh one on every visit)
- 🔄 **Live HEX ↔ RGB converters** — no buttons, results update as you type
- 📋 **Smart RGB paste** — paste `255, 0, 128` into any field and it auto-splits
- 📎 **One-click copy** for every value
- 🌗 **Automatic text contrast** (black/white) based on the background color
- ✨ **Animated, color-reactive UI** — the background reacts to the picked color
- 📱 Responsive layout
- ⚡ Static export (no backend)

## Local Development

```bash
npm install
npm run dev
```

Open **http://localhost:3000** — in development the app runs at the root.

> The `basePath` (`/colorchroma`) is applied **only in production builds**, so
> GitHub Pages keeps working while local dev stays on the root. See
> [`next.config.ts`](next.config.ts).

### Other scripts

```bash
npm run build     # static export into ./out
npm run preview   # serve the exported ./out locally
npm run lint
```

## Deployment

Pushing to `main` triggers [`.github/workflows/deploy.yml`](.github/workflows/deploy.yml),
which builds the static site (`output: 'export'`) and publishes `./out` to GitHub Pages.

The favicon ([`src/app/icon.svg`](src/app/icon.svg)) and the social preview image
([`src/app/opengraph-image.tsx`](src/app/opengraph-image.tsx)) are generated by
Next.js at build time — no manual image assets required.
